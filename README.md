Demo: <a href="https://yryrrf.github.io/Metronome/index.html">Here</a>


# Tutorial
Load javascript
```HTML
<script src="metronome.js"></script>
```


Initialization
```HTML
var metronome = new Metronome();
```

Start the metronome
```HTML
metronome.start();
```

Stop the metronome
```HTML
metronome.stop();
```

Change tempo
```HTML
metronome.tempo = 60; //set tempo value
```
